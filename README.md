# Customer Purchase Behavior Analysis

## Overview

This project demonstrates an end-to-end Data Analytics workflow using Python, PostgreSQL, and Power BI. The goal was to analyze customer purchasing behavior, identify trends, and generate actionable business insights through data exploration, SQL analysis, and interactive dashboards.

The project covers the complete analytics lifecycle, including data loading, cleaning, exploratory data analysis (EDA), database querying, visualization, reporting, and presentation creation.

---

## Dataset

The dataset contains customer shopping and purchasing information, including:

* Customer ID
* Age
* Gender
* Item Purchased
* Category
* Purchase Amount
* Location
* Shipping Type
* Review Rating
* Discount Applied
* Previous Purchases
* Payment Method
* Frequency of Purchases

The data was analyzed to understand customer behavior, product performance, and purchasing trends.

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Examined dataset structure and data types.
* Performed initial data validation.

### 2. Data Cleaning

* Checked for missing values.
* Removed duplicate records.
* Standardized data formats.
* Validated data consistency.

### 3. Exploratory Data Analysis (EDA)

Performed detailed analysis to understand:

* Customer demographics
* Purchase behavior
* Product popularity
* Category-wise performance
* Rating distributions
* Discount impact on sales

Visualizations were created using Matplotlib and Seaborn to identify trends and patterns.

### 4. SQL Analysis Using PostgreSQL

The cleaned dataset was imported into PostgreSQL for advanced querying.

Key business questions answered:

* What are the most purchased products?
* Which categories generate the highest sales?
* How do discounts affect purchases?
* What is the average purchase amount by category?
* Which shipping methods are most preferred?
* How can customers be segmented based on purchase history?

SQL concepts used:

* Aggregate Functions
* GROUP BY
* ORDER BY
* CASE Statements
* Common Table Expressions (CTEs)
* Window Functions
* Subqueries

## How to Run

### Prerequisites

Install the following tools:

* Python 3.x
* PostgreSQL
* Power BI Desktop
* Jupyter Notebook

### Clone Repository

```bash
git clone https://github.com/your-username/customer-purchase-behavior-analysis.git
```

### Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn psycopg2
```

### Run Analysis

1. Open Jupyter Notebook.
2. Execute the Data Cleaning notebook.
3. Run the EDA notebook.
4. Import the cleaned dataset into PostgreSQL.
5. Execute SQL queries for business analysis.
6. Open the Power BI dashboard file to explore visual insights.
